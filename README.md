# NBA Playoff Bracket Game

This is a GitHub Pages based "game" that lets a player predict the outcome of the NBA playoffs and then assigns a score to their prediction based on how accurate it ended up being.
There is currently no backend for this game; it is the burden of the player to save their bracket and refer back to it later.

## How to Play

To play the game, the player must create a bracket. They do this by going matchup by matchup, predicting the winner and how many games the series will last for. They do this until reaching the NBA Finals.
